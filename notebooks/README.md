# notebooks/

Optional exploratory / analysis notebooks (e.g. profiling the synthetic data, prototyping
AML risk-scoring features before implementing them in Snowflake SQL or Snowpark).

- All notebooks run on **synthetic data only** and require no real credentials.
- This folder is intentionally light in early phases; content is added where it adds value.
- Production-style logic lives in [`../snowflake/`](../snowflake); notebooks are for
  exploration and explanation, not the system of record.
